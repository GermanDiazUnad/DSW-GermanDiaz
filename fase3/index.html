Flex Box, CSS Grid


Este sitio tiene como propósito el aprendizaje y utilización de manera adecuada de las etiquetas Css Flex Box y Css Grid. Gracias a estas etiquetas permite la maquetación de sitios web de una manera mas sencilla y dinámica, dejando atrás los elementos flotantes que generaban dolores de cabeza a la hora de aplicar Medias Query para adaptar nuestra implementación a otras resoluciones.


Temas: En esta sección encontrará texto e imagenes de apoyo el cuál ilustrán para un mejor comprención sobre Css Grid y Flexbox.

Recusos digitales: En está sección encontrará dos postcast, los cuáles definirán cada uno de los temas.
